# Screen 4: Chat Widget

**Mode:** Web
**Priority:** Medium — the closer, drives booking

## Prompt

```
Design an AI chat panel that slides in from the right side of a dark SaaS app.

Header: Agent name, online status indicator (green dot), close button.

Messages: Assistant messages on the left with a small 'F' brand avatar. User messages on the right in a slightly lighter bubble. Messages should have generous spacing — not cramped.

Special elements inline in chat:
- A 'data card' showing a website screenshot thumbnail with annotation count
- A 'booking card' with a calendar icon and gold CTA button

Input: Textarea at bottom with a gold circular send button. Placeholder: 'Ask about your results...'

Style: dark navy panel, glassmorphism, clean typography. The chat should feel like talking to a smart consultant, not a chatbot.
```

## What we're fixing
- Messages are walls of text — need breathing room
- No visual distinction between user/assistant beyond alignment
- Data cards are just text labels — need actual screenshot thumbnails
- Opener message is one giant paragraph — needs multi-beat delivery
- No personality in the design
