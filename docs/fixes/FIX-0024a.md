# FIX-0024a: Cal.com embed in chat — AI Engine system prompt protocol

**Audit ref:** AUDIT-003, Phase A item 5
**Assigned to:** AI Engine
**Priority:** P2 (breaks user flow)
**Status:** pending

## Problem
The `calcom_embed` SSE event is defined in `contracts/events.ts:102`, and the frontend handles it (`ChatContainer.tsx:159-172` → `DataCard.tsx:14-35` → `CalcomModal`). The system prompt (`sales-agent-system.ts:113-118`) tells the AI to "suggest booking with this link" and mentions "the frontend will render a Cal.com embed," but provides no structured protocol for triggering the embed. The AI just outputs the URL as plain text, which the user must click manually instead of getting the premium inline booking modal.

## File(s) to modify
- `src/lib/prompts/sales-agent-system.ts` — around line 113-118: add explicit embed trigger protocol

## Required change
1. Update the `## CAL.COM BOOKING` section to define a marker protocol:
   - Format: `[CALCOM_EMBED]` — a simple flag marker (no parameters needed since the Cal.com URL is already configured)
   - Instruct the AI: "When you want to offer booking, include `[CALCOM_EMBED]` in your response. The frontend will render an interactive booking widget inline in the chat. Do NOT paste the raw URL — use the marker instead."
2. Define when to trigger:
   - After the AI has established rapport and the lead shows interest
   - When the lead explicitly asks about booking or pricing
   - After presenting the key findings (2-3 exchanges in)
   - Maximum once per conversation (don't spam the embed)
3. Remove or update the instruction to "suggest booking with this link: ${calLink}" — replace with the marker protocol

## Constraints
- Do NOT modify Frontend files — ChatContainer, DataCard, and CalcomModal already work
- Do NOT modify `contracts/events.ts` — the event type is correct
- Do NOT modify Backend stream API — FIX-0024b handles parsing
- The marker must be simple and unambiguous for regex parsing

## Verification

2. Confirming raw URL suggestion is replaced with marker instruction
